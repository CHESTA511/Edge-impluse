Selamat Datang di Edge

Sistem Monitoring Gempa Real-Time

Edge membantu Anda memantau aktivitas seismik secara langsung dari perangkat sensor. Gunakan navigasi di atas untuk melihat data real-time, memeriksa riwayat peristiwa, atau melihat lokasi sensor di peta.

Fitur Utama

Monitoring Real-Time

Lihat data sensor (vibrasi, pergeseran, sumbu) yang diperbarui setiap detik langsung dari perangkat Anda.

Riwayat & Replay

Akses log peristiwa penting (seperti "GEMPA KUAT!") dan putar ulang data historis sensor dalam bentuk grafik.

Peta Lokasi Interaktif

Visualisasikan lokasi perangkat Anda di peta interaktif dan lihat status terakhir mereka secara real-time.
