Selamat Datang di VoxSilva

Low-Cost AI-Powered Forest Surveillance System

VoxSilva adalah sistem pemantauan hutan berbasis AI untuk mendeteksi aktivitas illegal logging. Sistem ini menggunakan jaringan sensor yang memantau setiap titik di hutan dengan radius 1-3km per node. Pantau deteksi manusia, suara chainsaw, dan pergerakan perangkat secara real-time.

Analisis Sistem

Status Jaringan: Semua node berfungsi normal. Koneksi Base Station stabil.

Pemeliharaan: Node #7 memerlukan pengecekan baterai dalam 3 hari ke depan.

Perhatian: Peningkatan aktivitas terdeteksi di sektor timur. Disarankan patroli rutin.

Fitur Utama

Sensor Network

Pantau jaringan sensor Base Station dan Node yang tersebar di hutan. Setiap node memiliki radius cakupan 1-3km untuk deteksi illegal logging.

AI Detection System

Deteksi otomatis manusia, suara chainsaw, dan pergerakan perangkat menggunakan AI. Dapatkan rekomendasi dari asisten AI untuk pemeliharaan.

Real-Time Monitoring

Pantau data MPU6050 (x,y,z), deteksi visual ESP32 Cam, dan deteksi audio chainsaw secara real-time dari semua node.
